# mlrotator
A python image dataset creator, for use with training your image recognition models.
# Requirements 
You'll need to install Ursina first before using, simply running:
```
pip install ursina
```
in your terminal
# How to use
Import texture by linking the directory of the image file inside the input bar, set XYZ rotation/sec, and click Source (src). The images will be inside the output folder.
